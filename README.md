# Weather API

API REST desenvolvida em **Java 17** com **Spring Boot**, que fornece informações meteorológicas de forma simples, organizada e extensível.  
Projeto criado para fins de estudo e prática de desenvolvimento back-end.

# Tecnologias utilizadas

- Java 17+
- Spring Boot
- Spring Web
- Maven
- OpenWeather API (ou outra API de clima, conforme sua integração futura)

# Funcionalidades

- Buscar informações do clima por cidade  
- Retornar temperatura, umidade, sensação térmica e condições gerais (ex.: nuvens, chuva, etc.)  
- Padronização de respostas com DTOs  


# Como inicializar a API

- Faça um git clone do repositório
- crie um arquivo .env com a chave da API externa
- execute o projeto
